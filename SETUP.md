### üöÄ Quick Setup

#### Step 1: Create GitHub Profile Repository
Create a new repository on GitHub with the following settings:
- **Repository name**: `yourusername` (must match your GitHub username exactly)
- **Visibility**: Public
- **Description**: (optional) "My GitHub profile README"

#### Step 2: Upload Files
Upload the following files to the repository:
- `README.md`
- `jh_logo.svg`
- Optionally: `jh_logo_static.svg` (fallback)

#### Step 3: Customize Your Information
Edit `README.md` and replace these placeholders:

**Personal Info:**
- Replace `yourusername` with your GitHub username in all URLs
- Update social media links (LinkedIn, Twitter, Portfolio, Email)
- Modify the About Me section with your actual details

**Featured Projects:**
- Replace project descriptions with your actual projects
- Update tech stacks to match your real technologies
- Add links to your repositories

**GitHub Stats:**
- Replace `yourusername` in the stats URLs with your actual username
- If you want to hide private repositories, add `&count_private=false`

### üìù Typing Animation

The typewriter effect uses [readme-typing-svg](https://github.com/DenverCoder1/readme-typing-svg).

Parameters you can adjust:
- `lines` - Text to display
- `pause` - Pause between lines (milliseconds)
- `color` - Text color
- `size` - Font size
- `random` - Random order (true/false)

### üêõ Troubleshooting

**Stats not showing?**
- Make sure your username is correct
- Wait a few minutes for the cache to update
- Check that your repository is public

**Logo not animating?**
- Ensure you're using `jh_logo.svg` (animated version)
- GitHub may cache for a few minutes
- Try hard refresh (Ctrl+F5 / Cmd+Shift+R)

**Badges not rendering?**
- Check the Shield.io service is online
- Verify badge URLs are correctly formatted
- Ensure no special characters are breaking the URLs
